Code Log 4: CSS Color

How CSS Uses Color

In CSS, you can use color to control the visual appearance of specified sections of text, backgrounds, and borders on a webpage. Below are some ways in which you can implement color in CSS:

CSS Color Properties

  1. color — Defines the color of the text inside an element.
    Example: color: blue; changes the text color to blue.
  2. background-color — Sets the background color of an element’s content area.
    Example: background-color: lightgray; fills the element’s background with light gray.
  3. border-color — Determines the color of an element’s border.
    Example: border-color: black; makes the border black.

CSS Color Values

Here are some examples of defining colors:

  1. Named Colors: Use predefined names such as teal or navy.
    Named color example: background-color: teal
  2. Hexadecimal Colors: Represented with a # followed by six digits (0–9, A–F). Example: #ff6347 for tomato red.
    Hex color example: background-color: #ff6347
  3. RGB Colors: Defined by mixing red, green, and blue light values (0–255). Example: rgb(60, 179, 113) for medium sea green.
    RGB color example: background-color: rgb(60, 179, 113)
  4. HSL Colors: Defined using hue, saturation, and lightness percentages. Example: hsl(210, 100%, 50%) for bright blue.
    HSL color example: background-color: hsl(210, 100%, 50%)
