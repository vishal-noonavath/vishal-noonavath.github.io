Code Log 3: CSS Media Queries

How does a CSS media query work, and why is it useful?

A CSS media query allows you to apply specific CSS styles based on the type of device you're viewing it from, such as the width, height, orientation, or resolution. It works like a conditional statement in CSS: if the media query condition is true, then the styles inside it will be applied.

This is an example of responsive web design as it helps your code adapt its layout to different devices without needing multiple versions of the same webpage. This way, it ensures that viewers have an similar/optimal viewing experience regardless of how they view the webpage.

For example: 

            @media screen and (max-width: 600px) {
                body {
                    background-color: lightgray;
                }
            }

In the example, the background color will only change to light gray if the screen width is 600 pixels or less. This is an example of how you can implement flexible design adjustments based on device width.

What "media features" are supported by media queries, and what do they do?

Media features are the conditions that media queries check to determine whether to apply certain CSS rules. Some examples include min-height, min-width, orientation, resolution, and hover. Here are some examples:

  1. min-height
    • This feature applies styles when the browser’s viewport height is greater than or equal to the specified value. It’s useful for ensuring that certain layouts or elements adjust when the vertical space on the screen increases.
    • Example: @media (min-height: 700px) applies styles only when the viewport height is 700 pixels or taller.
  2. min-width
    • The opposite of max-width, this feature applies styles when the viewport width is greater than or equal to the specified value. It’s commonly used for larger screens, such as tablets or desktops.
    • Example: @media (min-width: 1024px) targets screens that are at least 1024 pixels wide.
  3. orientation
    • This feature checks whether a device is in portrait (taller than wide) or landscape (wider than tall) mode. It’s often used for mobile and tablet layouts that change when a user rotates their device.
    • Example: @media (orientation: landscape) applies styles only when the device is held horizontally.
  4. resolution
    • This feature detects the pixel density of a screen, allowing developers to serve higher-quality images or sharper visuals to high-resolution displays like retina display on mac devices.
    • Example: @media (min-resolution: 2dppx) applies styles to screens with a pixel density of 2 or higher.

CSS media queries are an essential part of responsive design, enabling developers to create accessible, and user-friendly websites that adjust seamlessly across all devices and screen sizes.
